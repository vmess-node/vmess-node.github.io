小火箭全攻略：新时代科学上网的钥匙

在信息高速传播的当下，获取真实、全面、自由的信息已成为每个人的需求。而在网络封锁与审查仍然存在的环境下，“科学上网”成为不少人通往开放互联网世界的通道。在这其中，“小火箭”（Shadowrocket）作为一款功能强大、使用便捷的工具，迅速脱颖而出，成为众多用户的首选。本文将从原理讲解、功能剖析、配置指南，到注意事项和常见问题，全面解析小火箭如何助你轻松实现科学上网。

一、什么是小火箭（Shadowrocket）？

小火箭是一款基于iOS平台开发的网络代理客户端工具，最初由国外开发者推出，专为搭配Shadowsocks、V2Ray等代理协议使用。尽管最初只支持iOS平台，但目前已经有Windows、Mac和Android上的类似替代品（如Shadowrocket for Windows、Streisand等），使得更多用户能够跨平台体验其便利。

本质上，小火箭就是一个中介，它接管你的部分或全部网络流量，通过设置好的代理服务器绕过防火墙，从而访问被限制的网站和应用。

二、小火箭的核心特点

小火箭之所以能在众多科学上网工具中脱颖而出，靠的是其强大、灵活且用户友好的设计。以下是它几个最关键的优势：

1. 多协议支持

支持Shadowsocks、Vmess、Trojan等多种协议，兼容性极强，适配不同类型的节点。

2. 自定义规则灵活

支持自定义分流规则（如ACL、GFW List、自定义正则等），可以设置哪些应用或网站走代理，哪些走直连，极大提升效率。

3. 实时日志与测速功能

用户可以实时查看代理连接状态、访问日志和流量速度，便于调试与优化。

4. 高度加密，保护隐私

采用TLS、AES等高强度加密协议，确保传输数据的安全性，防止被监控或劫持。

5. 配置导入便捷

支持二维码扫描、一键导入、URL订阅，用户无需手动输入繁琐参数。

三、科学上网的现实意义

1. 获取未被审查的信息

在一些网络受限地区，许多国际新闻、社交媒体、搜索引擎等无法访问，小火箭提供了通向真实世界的通道。

2. 保障数据隐私

在公共Wi-Fi环境中，使用小火箭等加密代理能够有效防止数据被窃取。

3. 扩展知识与视野

通过科学上网，你可以自由访问全球的学术资源、文化内容、技术论坛，从而获得更广阔的学习和发展空间。

五、小火箭详细配置指南

小火箭的使用并不复杂，但首次配置时可能会让新手感到困惑。下面是一个从零开始的简明教程：

步骤一：准备代理节点

你需要一个可以用的代理节点，来源包括：

  • 购买付费机场服务（稳定、安全）；

  • 获取免费节点（速度慢，适合测试）；

  • 从开源社区或朋友处获取。

节点一般以二维码、订阅链接、或JSON格式提供。

步骤二：添加节点

打开小火箭后：

  • 点击右上角“+”号；

  • 扫描二维码或粘贴节点链接；

  • 确认导入成功，节点将出现在服务器列表中。

步骤三：设置规则

进入小火箭设置：

  • 选择模式：建议使用“自动”或“配置分流”；

  • 编辑规则：可使用默认的GFW列表，也可手动添加要代理或绕过的网址；

  • 开启“连接日志”和“测速功能”，便于观察运行状态。

步骤四：开启代理

  • 连接你导入的节点；

  • 在设置中开启“小火箭代理”（需VPN权限）；

  • 验证连接是否生效（打开被墙网站即可）。

FAQ

Netch 的全局代理与规则代理有何不同？
全局代理将所有流量都通过节点转发，而规则代理只针对指定 IP、端口或应用。全局代理适合游戏和全局科学上网，规则代理适合优化带宽和分流特定流量。
Trojan 服务器如何防止被扫描到真实 IP？
可以把 Trojan 部署在反向代理或 CDN 后端，使用 CDN 的中转与域名伪装；同时避免在服务器上暴露其他与域名不匹配的服务，关闭 FTP/SSH 直连并设置防火墙白名单。
PassWall2 在路由器上使用 TUN 模式会有什么影响？
启用 TUN 模式会拦截并转发路由器上全部或选定流量，这会增加路由器 CPU 与内存负担，可能影响路由器其他服务表现。建议在性能足够的设备上启用，或仅对特定设备/端口进行分流。
Hysteria2 的 QUIC 协议相比 TCP 有什么优势？
QUIC 协议基于 UDP，减少了握手延迟，支持多路复用和拥塞控制优化。Hysteria2 利用 QUIC 提供更低延迟、更高稳定性的网络连接，尤其适合高延迟或丢包环境。
Potatso Lite 如何结合手动节点选择优化访问？
用户可根据网络延迟和用途手动切换节点，选择最优节点处理不同流量。虽然不支持自动更新，但在节点不稳定或特定需求下可以灵活调整，提高访问效率。
SSR 的协议插件如何应对封锁？
协议插件可隐藏真实通信特征，将流量伪装成正常 HTTP 或 TLS 数据。结合混淆方式使用，可以有效降低被 DPI 检测或封锁的风险，提高节点稳定性和长期可用性。
Clash Premium 与普通版本有何不同？
Clash Premium 是 Clash 的增强版，增加了 TUN 模式、脚本功能、流量统计与规则订阅等高级特性。它更适合有复杂分流需求的用户，尤其是需要对系统级代理进行全局控制的场景。
Surfboard 可以结合规则实现广告屏蔽吗？
可以。Surfboard 支持通过规则文件匹配广告域名或 URL，将其阻断或重定向。结合策略组定期更新规则，可以实现自动广告屏蔽，提升网页浏览体验并节省带宽。
为什么某些网站在通过代理后登录会频繁要求二次验证？
代理会改变访问 IP 与地理位置，目标站点可能检测到异常登录并触发验证码或二次验证。建议对常用账户使用白名单 IP 或在代理配置中对特定站点设置直连来避免频繁验证。
SSR 协议的多路复用功能如何提升性能？
SSR 多路复用允许多个短连接共享同一通道，减少握手次数和加密开销。适合游戏、即时通讯等高频短连接场景，提升网络吞吐量和连接稳定性，提高整体上网体验。
